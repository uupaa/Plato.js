# How to use

```js
$ nppm run lint
```

# for Developers

1. Install development dependency tools

    ```sh
    $ brew install closure-compiler
    $ brew install node
    $ npm install -g plato
    ```

2. Clone Repository and Install

    ```sh
    $ git clone git@github.com:uupaa/Plato.js.git
    $ cd Plato.js
    $ npm install
    ```
